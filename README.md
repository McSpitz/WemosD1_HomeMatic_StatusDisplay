## Zusammenfassung:
Ziel dieses Projektes ist der kostengünstige Bau einer Statusanzeige, ähnlich der von HomeMatic angebotenen.

Die Helligkeit der LEDs lässt sich mit einem Taster in 25 Stufen einstellen.
